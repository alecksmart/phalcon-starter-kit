# phalcon-starter-kit
A custom starter boilerplate with php7, phalcon framework and live reload

# Pre-requisites

 * npm
 * composer
 * phalcon extension for php

# Install

```bash
$ npm install
$ composer install
$ ln -s 'PATH/TO/vendor/bin/phalcon.php' 'bin/phalcon'
# check with:
$ bin/phalcon commands
```
